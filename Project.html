Overview:

Understanding HTML5 is a necessary skill for those who are creating and maintaining Web sites. Being able to show a perspective client or employer a well-organized Web site is critical to getting work in this field.

To show an understanding of HTML5 and as a practice for building a Web site in a realistic situation this assignment is beneficial to the student.

Purpose:

The purpose of authentic learning activity is to evaluate each student's ability to prepare and design a web site relating to a central theme.

At the conclusion of this learning activity, students should be able to plan, create, develop and save a Web site for review by a perspective client or employer.

Standard:

The student must make at least 70% on the project to pass the course.

Requirements:

You will create a personal Web site. If you already have a personal Web site you can create any Web site you like as long as it is related to you in some manner. You must use HTML5 code to create this site, use of any other Web design software (e.g. Dreamweaver) is strictly forbidden and will result in a 0 (zero) grade for this project.

You need to include the following information in your Home page.

  • This is my CIST 1510 course project
  • Your name
  • Date you completed the project
  • A copy of the school's logo that is linked to the school's Web site, near the bottom of the home page

On your Web site:

You need to create your pages to include the following:

  • graphic elements
  • hyperlinks
  • tables
  • image maps
  • CSS
  • forms
  • multimedia techniques
  • a reflection statement on this project
  • a completed rubric for this assignment

You need to create a home page and at least 4 pages other pages that are linked to the other pages in this Web site.

A sentence of warning! Do not wait until the last minute to do this project. It is intended to be an ongoing project to be worked on throughout the quarter!
